# Estrutura Ideal de Bug Report
Título: [Módulo] Descrição curta e direta

Ambiente: Versão / Navegador / OS

Severidade: Crítico / Alto / Médio / Baixo

Comportamento Esperado:
Descreva o que o sistema deveria fazer.

Comportamento Atual:
Descreva o que está acontecendo de errado.

Passos para Reproduzir:
1. Acesse...
2. Clique em...
3. Observe que...

Evidência: Print / Vídeo anexado
⚠ Evite no título:
❌ "Está quebrando"
❌ "Bug no sistema"
❌ "Não funciona"
✅ Prefira:
✅ "[Estoque] Qty não decrementa após confirmar saída"
✅ "[Produto] Cadastro aceita preço negativo sem validação"
